# GPUSketch Figures

This directory contains scripts for generating figures in the paper.

### Scripts
- `cpu_gpu_sketch_throughput.py`: Throughput comparison between CPU vs. GPU for sketch updates.
- `minimum_cut_approx_box.py`: A box plot of minimum cut approximation results of kron17 and ktree17 dataset.
- `standalone_perf.py`: A standalone performance of each subsystem, used for performance analysis.
